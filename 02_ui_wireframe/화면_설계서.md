# 화면 설계서 (Screen Design UI/UX)

## 1. 공통 레이아웃 (Layout)

### 1.1 네비게이션 & 사이드바
- **Top Navbar**
  - 로고 및 서비스 명 (클릭 시 Home 이동)
  - Theme Switcher 아이콘 (☀️ 라이트 / 🌙 다크 전환)
  - 우측: 사용자 정보 (로그인 계정) 및 로그아웃 버튼
- **Sidebar (좌측)**
  - 네비게이션 메뉴:
    1. 대시보드 (Home)
    2. AI 애널리스트 채팅 (Chatbot)
    3. 투자 리포트 생성 (Report)
    4. 실적 캘린더 (Calendar)
  - **관심 기업 (Watchlist) 위젯**:
    - 리스트 형태의 저장된 티커 목록 (삭제 `X` 버튼 포함)
    - 입력 필드 및 추가(+) 버튼: 한글 구어체(예: "애플", "엔비디아") 입력 시 티커(AAPL, NVDA) 자동 치환 기능 및 피드백 메시지 노출 영역
- **Page Transition**
  - 메뉴 간 이동 시 화면 전환 대기 시간을 감추기 위한 전체 화면 로딩 오버레이("페이지 이동 중...")

## 2. 화면 상세 설계

### 2.1 대시보드 (Dashboard / Home - `home.html`)
- **목적**: 접속 시 시장의 전반적인 요약 정보와 사용자 중심의 지표를 한눈에 파악
- **구성 요소**:
  - **환율 정보 카드 (Top)**: 실시간 KRW/USD 환율 (`Exchange Rate API 연동`) (상승/하락 색상 배지)
  - **현재 DB 상태 요약 카드**:
    - S&P 500 등 수집된 기업 수, 분석된 문서 수
  - **데이터 시각화 (Charts Area)**:
    - 2단 분할 레이아웃 (Bootstrap Grid)
    - [좌측] 시가총액/매출 상위 10개 기업 Bar 차트 (Plotly.js)
    - [우측] 모니터링 중인 섹터(Sector) 분포 Pie/Donut 차트 (Plotly.js)
  - **최근 활동 내역 (Recent Activity)**: 하단에 최근 챗봇 질문 히스토리 카드 목록으로 노출 클릭 시 해당 답변 내용 모달 팝업

### 2.2 AI 애널리스트 챗봇 (`chat.html`)
- **목적**: 하이브리드 RAG (Vector + Graph) 기반의 대화형 정보 탐색
- **구성 요소**:
  - **대화 영역 (Chat View)**:
    - 스크롤 가능한 메시지 창 (사용자 프롬프트 우측 정렬, AI 답변 좌측 정렬)
    - AI 답변 내부에 마크다운 파싱 렌더링 적용 (표, 코드 등 표현)
    - AI가 답변을 생성할 때 참조한 데이터의 출처(Sources)를 Tooltip 또는 접기/펴기 형태로 제공
  - **입력 폼 (Bottom Sticky)**:
    - 텍스트 텍스트 에어리어 (멀티라인 입력 지원)
    - 전송 버튼 (로딩 중 비활성화 처리)
  - **추천 질문 (Quick Actions)**:
    - 채팅창 상단 또는 빈 화면일 때 제공되는 칩(Chip) 버튼들 (예: "애플의 최근 실적은?", "엔비디아와 TSMC의 관계 요약해줘")
    - 클릭 시 입력 필드로 텍스트 삽입

### 2.3 투자 리포트 생성 (`report.html`)
- **목적**: 상세한 기업 분석 및 종목 간 재무/시장 지표 비교 레포트 제공
- **구성 요소**:
  - **컨트롤 패널 (Top)**:
    - 타겟 기업(Target Company) 선택 Dropdown 또는 Input
    - 비교 대상 기업(Compare with) 선택 (Opitonal)
    - '분석 리포트 생성' 실행 버튼
  - **시각화 뷰어 (Chart Area)**:
    - 탭(Tab) 메뉴: 1) 주가 추이 (캔들 차트), 2) 거래량, 3) 주요 재무 지표 비교
    - 동적 Plotly 차트 렌더링 및 Zoom/Pan 컨트롤
  - **분석 텍스트 결과 영역**:
    - LLM이 생성한 마크다운 형태의 상세 리포트 덤프 영역
  - **내보내기 (Export)**: 우측 상단 'PDF로 저장' 버튼 제공

### 2.4 실적 캘린더 (`calendar.html`)
- **목적**: Watchlist에 등록된 기업들의 실적 일정 확인 및 서프라이즈 여부 직관적 파악
- **구성 요소**:
  - **필터 영역**:
    - 해당 연도 설정 (Input: 2024, 2025 등)
    - 분기(Quarter) 라디오 버튼 (1Q, 2Q, 3Q, 4Q)
    - '조회' 버튼 (로딩 상태 스피너 지원)
  - **결과 리스트 영역**:
    - **빈 화면 상태**: "관심 기업이 등록되어 있지 않습니다" 또는 "조회된 일정이 없습니다" 안내 UI 노출
    - **결과 노출 상태**:
      - 날짜별(아코디언 형태)로 그룹핑된 리스트. 각 그룹을 클릭하여 펼치기/접기 가능
      - 행렬 테이블(Table) 형태 데이터: 티커명, 예상 EPS, 실제 EPS, 서프라이즈 비율
      - 서프라이즈 비율에 따른 시각적 배지: 예측치 상회(Green/↑), 하회(Red/↓) 적용

### 2.5 AI 파이프라인 연동 화면 (Model Integration)
- **FinBERT 감성 분석 데이터 연동**: 챗봇 하이브리드 RAG에 FinBERT 데이터가 통합되었으며, 사용자가 특정 기업을 질의할 때 관련된 공급망 및 경쟁사의 뉴스 감성 지수(Positive/Negative)를 AI 답변 내의 컨텍스트(소스)로 자연스럽게 녹여내어 제공한다.
- **AI 예측 대시보드 (추후 편입 예정)**: ML Quant 모델의 주가/재무 예측치 파이프라인이 완료되면, 향후 대시보드 화면에 '예측 기반 차트 영역'으로 시각화하여 확장할 예정이다.
